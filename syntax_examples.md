# Loong 解释器语法示例

## 变量赋值
```plaintext
a = 10
b = 3.14
```

## 字符串
```plaintext
c = "Hello, " + "world!"
```

## 条件运算符
```plaintext
result = (a + b > 12 ? "Yes" : "No") + ", " + c
```

## 函数定义
```plaintext
fun sum(x, y):
    x + y
end
```

## 函数调用
```plaintext
result = sum(a, b)
```

## 列表
```plaintext
my_list = [1, 2, 3, 4, 5]
first_element = my_list[0]  # 访问第一个元素
my_list.append(6)            # 向列表添加元素
```
